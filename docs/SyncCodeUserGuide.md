# 同步码使用指南

## Summary

同步码用于在不同设备、不同浏览器之间同步同一份首页配置。你可以在一台设备上创建同步码，再把这个同步码输入到另一台设备中。绑定后，两边可以通过手动“上传”和“拉取”交换最新的首页内容。

当前阶段是 MVP 手动同步模式：修改首页后需要点击“上传”，另一台设备需要点击“拉取”才能获得最新内容。

## 基本概念

- **本地首页**：当前浏览器里的首页数据，保存在当前浏览器本地。
- **同步码**：用于绑定同一个云端同步空间的凭证。
- **上传**：把当前浏览器里的首页内容保存到云端。
- **拉取**：从云端读取最新首页内容，并覆盖当前浏览器里的首页。
- **解除本机**：当前浏览器不再绑定同步码，但云端数据仍保留。
- **废弃同步码**：让这个同步码彻底失效，所有设备都不能继续使用它同步。

## 在第一台设备上创建同步码

1. 打开本地或线上首页。
2. 点击右上角 `编辑`。
3. 找到 `同步码` 区域。
4. 点击 `创建`。
5. 页面会生成一串同步码。
6. 点击 `复制`，并把同步码保存到安全位置。

创建完成后，当前浏览器已经绑定这个同步码。此时当前首页内容已经上传到云端一次。

## 在第二台设备或浏览器上绑定同步码

1. 在另一台设备、另一个浏览器，或无痕窗口中打开首页。
2. 点击右上角 `编辑`。
3. 找到 `同步码` 区域。
4. 把第一台设备复制的同步码粘贴到 `输入同步码`。
5. 点击 `绑定`。
6. 浏览器会提示：绑定会用云端首页覆盖当前本地首页。
7. 确认后，当前设备会拉取云端首页并完成绑定。

绑定成功后，这台设备就和第一台设备使用同一个同步空间。

## 在一台设备修改后同步到云端

假设你在设备 A 上修改首页内容：

1. 打开设备 A 的首页。
2. 点击 `编辑`。
3. 新增、编辑、删除或排序分组和网站。
4. 确认页面显示已保存。
5. 在 `同步码` 区域点击 `上传`。
6. 页面提示上传成功后，云端就是设备 A 的最新版本。

注意：只修改本地内容但不点击 `上传`，其他设备不会看到这些改动。

## 在另一台设备获得最新改动

假设设备 A 已经上传了最新内容，现在要让设备 B 获取这些改动：

1. 打开设备 B 的首页。
2. 点击 `编辑`。
3. 在 `同步码` 区域点击 `拉取`。
4. 页面会从云端读取最新首页。
5. 拉取成功后，设备 B 的首页会被云端版本覆盖。

注意：`拉取` 会用云端数据覆盖当前浏览器本地数据。如果设备 B 上有未上传的本地修改，建议先确认是否需要保留。

## 推荐使用流程

### 单设备修改，多设备查看

1. 平时只在一台主设备上编辑首页。
2. 修改后点击 `上传`。
3. 其他设备需要更新时点击 `拉取`。

这是当前阶段最稳定的使用方式。

### 多设备都可能修改

如果多台设备都会修改首页，建议遵守这个顺序：

1. 修改前先点击 `拉取`，确保当前设备是最新版本。
2. 修改首页内容。
3. 修改完成后点击 `上传`。
4. 换到另一台设备时，先点击 `拉取` 再继续修改。

这样可以降低冲突概率。

## 冲突情况

当前版本已经能检测基础冲突，但还没有完整的自动合并能力。

常见冲突场景：

1. 设备 A 和设备 B 都绑定了同一个同步码。
2. 两台设备都没有先拉取最新版本。
3. 设备 A 修改后点击 `上传`。
4. 设备 B 也修改并点击 `上传`。
5. 设备 B 可能看到冲突提示。

遇到冲突时，当前建议：

1. 如果设备 B 的修改不重要，点击 `拉取`，使用云端版本。
2. 如果设备 B 的修改更重要，先手动记录改动，再拉取云端版本，然后重新修改并上传。

后续版本会增加更清晰的冲突处理界面。

## 解除本机绑定

如果你只想让当前浏览器停止使用同步码：

1. 点击 `编辑`。
2. 找到 `同步码` 区域。
3. 点击 `解除本机`。
4. 确认操作。

解除后：

- 当前浏览器会回到本地模式。
- 当前浏览器里的首页内容仍然保留。
- 云端同步空间不会删除。
- 其他已绑定设备仍然可以继续同步。

## 废弃同步码

如果你认为同步码已经泄露，或者不再希望任何设备继续使用它：

1. 点击 `编辑`。
2. 找到 `同步码` 区域。
3. 点击 `废弃同步码`。
4. 确认操作。

废弃后：

- 这个同步码会失效。
- 所有设备都不能继续用它上传或拉取。
- 如果还需要同步，需要重新创建新的同步码。

## 安全注意事项

- 同步码等同于该首页的读取和编辑权限。
- 不要把同步码发给不可信的人。
- 不要把同步码发到公开聊天、公开仓库或网页中。
- 普通首页网址本身不具备编辑权限。
- 如果怀疑同步码泄露，应立即点击 `废弃同步码`，然后重新创建。

## 当前限制

- 当前版本需要手动点击 `上传` 和 `拉取`。
- 当前版本不支持自动实时同步。
- 当前版本不支持多用户权限管理。
- 当前版本不支持只读分享链接。
- 当前版本不支持自动合并多设备同时编辑产生的冲突。

这些能力会在后续登录同步和会员功能阶段继续完善。
